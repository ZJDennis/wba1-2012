    <script src="jquery-1.8.3.min.js"></script>
</head>

Methode 2: Content Delivery Network einbinden 

<head>
    <script type=”text/javascript” src="//ajax.googleapis.com/ajax/libs/jquery/1.8.3/
    jquery.min.js"></script>
</head>

Hinweis: Um Versionsübergreifend gültig zu sein, kann man anstatt 1.8.3 auch nur 1 schreiben. Dann wird die aktuellste Version zwischen 1.0.0 und 1.9.9 eingebunden.

Let's start writing less

Syntax (1)

$(selector).action()

$ := jQuery-Notation; selector := CSS-Selector; action := jQuery-Funktion 

$("p").hide() - Blendet alle <p> Elemente aus.

$("#test","p").hide() - Blendet Elemenete mit der id "test" und alle <p> Elemente aus.

Basisfunktionen

Add und Remove

  • append() - Fügt Inhalt am Ende des ausgewählten Elements ein
  • prepend()- Fügt Inhalt am Anfang des ausgewählten Elements ein
  • before() - Fügt Inhalt vor dem ausgewähltem Element ein
  • after() - Fügt Inhalt hinter dem ausgewähltem Element ein
  • remove() - Löscht das ganze Element und die Kind-Elemente
  • empty() - Löscht nur die Kind-Elemente aus dem ausgewähltem Element

Übung 3

Die Dateien jquery.roundabout.js, jquery.roundabout-shapes.js und reflection/reflection.js müssen eingebunden werden.

Roundabout:


$('ul').roundabout();	// Für das Standard-Coverflow
$('ul').roundabout({shape: 'waterWheel' });
// Für andere Animationen z.B. figure8,square oder lazySusan.

Um die Reflexion der Bilder zu erzeugen muss lediglich die Klasse "reflect" den Bildern zugeordnet werden.


<img src="bild.jpg" class="reflect" />
