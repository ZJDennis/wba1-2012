1. Box-Modell

Zwei wichtige Zweige:

  • Box-Modell selbst
  • Element-Typen
    • block
    • inline

1.1. Grundlagen

alle Elemente im Rahmen von CSS sind entweder:

  • rechteckige Kästen (Block-Element)

    • oder

  • eine Art Streifen (Inline-Element)

1.2 Aufbau einer Box

  • Breite (width)
  • Höhe (height)
  • Äußenabstand (margin)
  • Rahmen (border)
  • Innenabstand (padding)

1.2.1 Aufbau einer Box (Beispiel)

#kasten1 {
	width: 300px;
	height: 200px;
	margin: 10px ;
	border: solid 6px black;
	padding: 20px;
}
<p id="kasten"> Lorem ipsum ... </p>

1.3 Element-Typen

  • Block-Element
    • beginnen in einer neuen Zeile
    • darauf folgenden Elemente ebenfalls
    • für sie gelten width, height, margin, border und padding
  • Inline-Element
    • erzeugen keine neue Zeile sondern stehen im Textfluss
    • width, height, margin gelten für sie nicht
